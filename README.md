## Installation:
1. Clone this repo:
```
$ git clone 
```

2. Install npm:

```
$ npm install
```

3. Run the project:
```
$ npm run start 
```

4. Open up the project in localhost

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
